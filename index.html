    Trebla

    A programming language and compiler for creating rhythm game engines on Sonolus.

    Quick Start

    1. Download the corresponding image for your operating system from releases.
    2. Extract the contents of the archive and add the bin folder to your PATH if desired.
    3. See guides or resources for more information.

    Building from Source

    Trebla was built and tested with JDK 14, but JDK 8 or higher should suffice.

    1. Clone via git. 
      git clone
    2. Run via gradle. 
      gradlew run --args="[arguments]"
      See gradlew tasks for other options.
